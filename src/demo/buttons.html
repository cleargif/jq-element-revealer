The Publishers

These buttons control the paragraphs below. Click them and see the subscribers react.
The buttons will publish a single event with a single value. It is the subsribers who listen for the event and the values that will make them show / hide.


<!-- Publishers -->
<div class="jqr-delegate-hook">
  <button type="button" class="jqr-publisher"
          data-jqr-publish-events="/buttons/event/"
          data-jqr-publish-value="button.attr.val1">
      ...
  </button>

  <button type="button" class="jqr-publisher"
          data-jqr-publish-events="/buttons/event/"
          data-jqr-publish-value="button.attr.val2">
      ...
  </button>
</div>
